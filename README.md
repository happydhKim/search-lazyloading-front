# search-place-front

## 프로젝트 소개🦉

카카오프렌즈 쇼핑몰의 외형을 클론하는 사이드 프로젝트입니다.  
기본적인 리스트를 레이지로딩과 검색 그리고 반응형 웹을 구현합니다.
